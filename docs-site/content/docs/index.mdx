**Brainly** is a "second brain" personal knowledge management app. Save any URL
(YouTube, Twitter/X, GitHub, articles, etc.), auto-enrich it with metadata, tag
it, and optionally share your whole brain publicly via a short link.

This site is the complete engineering reference for the monorepo — architecture,
backend internals, the full API, the frontend, testing, operations, and the
forward-looking AI roadmap.

## The stack at a glance

| Layer | Technology |
| --- | --- |
| **Backend** | Node.js + Express 4 + TypeScript, MongoDB via Mongoose 8, Zod, Pino |
| **Frontend** | React 19 + Vite 7 + TypeScript SPA, React Router 7, Tailwind CSS v4 |
| **Auth** | JWT + bcrypt, Google OAuth |
| **Testing** | Vitest, supertest, mongodb-memory-server, jsdom |
| **CI** | GitHub Actions (parallel backend + frontend jobs) |

## Core capabilities

- **Paste-to-save** — paste any URL; the system auto-detects the platform
  (YouTube, Twitter/X, GitHub, Medium, Instagram, or generic). A Notion provider
  exists but is disabled (no extractor yet — see
  [Functionality Fixes](/docs/changelog/functionality-fixes)).
- **Background enrichment** — a polling service fetches metadata (titles,
  descriptions, thumbnails, transcripts, README content) from platform APIs.
- **Tagging** — organize content with user-scoped tags.
- **Public sharing** — share your entire brain via a short read-only link.
- **Dual auth** — Google OAuth and local username/password.
